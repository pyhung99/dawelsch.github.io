Policy-Managed Microservices

Manage an infrastructure cluster as a single system to simplify container operations.
Try It Out

Contiv is a set of tools for managing policies to deploy microservices.

Contiv encodes operational intent in policies that can be applied to deployments of microservice-based applications, enabling network and storage solutions to be deployed uniformly and efficiently.

Contiv Network

Create network policy at the tenant, group, and application level.

Contiv Storage

Attach storage to containerized applications.

Contiv Cluster

Scale applications seamlessly and automatically.
